⚠️ A social media platform just like twitter, built using MERN stack and related technologies.

Key features and objectives:

- Users can create profiles, upload pictures and posts.
- Users can connect with each other, search for other users by name, view each other's profiles, and send friend requests.
- Users can comment on each other's posts and like them.
- Implement JWT (JSON Web Tokens) for authentication.

Frontend uses:

1. React
2. Redux
3. Vite
4. Styled components
5. Mantine

Backend uses:

1. Express
2. NodeJS
3. JWT Authentication
